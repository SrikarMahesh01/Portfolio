Publications PDF Upload Instructions

To enable the PDF viewer functionality for your publications, please follow these steps:

  1. Create a PDF file containing your complete list of publications
  2. Name the file exactly as: publications.pdf
  3. Place the file in this directory (public folder)

Recommended PDF Content:

  • Include a comprehensive list of all your publications
  • Organize them by type (IEEE, Scopus, etc.)
  • Include full citation details
  • Add DOI links where available
  • Use a clear, readable font and formatting

Important: The file must be named 'publications.pdf' for the viewer to work correctly.
